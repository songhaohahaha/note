### 4.标签：

​		<>用尖括号括起来的

​		双标签：成对存在（开始标签和结束标签）

​		单标签：只存在一个

### 5.标签构成：

​		 标签名 属性 = “属性值”

### 6.注释标签：


​		标签选择器：css：标签名

```

```


​		id选择器（唯一的）：

```
html： id=“box”    、   css：#id{ }
```

​	 返回一级是一个../
